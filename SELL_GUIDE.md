## 🚀 CÓMO MOSTRAR LA DEMO (PASO A PASO)

### **Opción A: En TU computador (recomendado)**

```bash
# 1. Abre terminal en la carpeta
cd "C:\Users\Estudiante\Desktop\web relojes"

# 2. Instala dependencias si no lo has hecho
npm install

# 3. Inicia el servidor
npm run dev
```

**Abre en navegador:** `http://localhost:3000`

**Para mostrar en tu celular:**
1. En terminal, busca tu IP:
   ```bash
   ipconfig | findstr "IPv4"
   ```
2. Verás algo como: `192.168.1.100`
3. En tu celular (conectado al mismo WiFi), abre: `http://192.168.1.100:3000`
4. ¡Listo! Puedes pasarle el teléfono al cliente.

---

### **Opción B: Enlace temporal (Replit/Vercel Preview)**

Si quieres que el cliente vea desde su casa:

1. **Vercel Preview** (gratis):
